Learning Goal Reminder

Track learning goals with progress milestones, deadlines, and escalating reminders

What this flow does

  1. **Tracks learning goals** with structured milestones and deadlines
  2. **Calculates progress** as a percentage based on completed milestones
  3. **Escalates reminders** as deadlines approach (on track → soon → urgent → overdue)
  4. **Highlights next milestones** so you always know what to work on next

How it runs

1. load-goals
file-system
2. goal.name || "Unknown",
evaluate
3. format-report
evaluate
4. notify
notify

Inputs & configuration

goals_file
Path to JSON file with learning goals, milestones, and deadlines · default: ~/.learning-goals.json
lookahead_days
Days ahead to warn about upcoming milestones · default: 7

README configuration

config:
  goals_file: "~/.learning-goals.json"
  lookahead_days: 7

Schedule & output

Cron: 0 9 * * *

Lobster workflow

View workflow YAML
# Lobster workflow: learning-goal-reminder
# Track learning goals with progress milestones, deadlines, and escalating reminders
#
# Usage:
#   lobster run --file workflow.yaml --args-json '{"goals_file":"/path/to/goals.json"}'
#
# Goals file format (JSON):
#   {
#     "goals": [
#       {
#         "name": "Learn Rust",
#         "deadline": "2026-06-01",
#         "milestones": [
#           {"title": "Complete Rust Book Ch1-5", "due": "2026-04-01", "completed": true},
#           {"title": "Build CLI tool project", "due": "2026-04-15", "completed": false},
#           {"title": "Contribute to open-source Rust project", "due": "2026-05-15", "completed": false}
#         ]
#       }
#     ]
#   }
#
# Requires: jq (>= 1.6 for strptime/mktime), date (GNU coreutils)

name: learning-goal-reminder
description: Track learning goals with progress milestones, deadlines, and escalating reminders

args:
  goals_file:
    description: "Path to JSON file with learning goals and milestones"
  lookahead_days:
    description: "Days ahead to check for upcoming milestones and deadlines"
    default: "7"

steps:
  - id: load-goals
    command: |
      if [ ! -f "${goals_file}" ]; then
        echo "{\"error\":\"Goals file not found: ${goals_file}\"}" >&2
        echo '{"goals":[]}'
        exit 1
      fi
      cat "${goals_file}"

  - id: evaluate-progress
    stdin: $load-goals.stdout
    command: |
      tmpfile=$(mktemp /tmp/lb_goals_XXXXXX.json)
      trap 'rm -f "$tmpfile"' EXIT
      cat > "$tmpfile"
      today=$(date +%Y-%m-%d)
      lookahead=${lookahead_days}
      jq -c --arg today "$today" --argjson look "$lookahead" '
        # Exact days between two YYYY-MM-DD dates using epoch seconds
        def days_between($d1; $d2):
          try (
            ($d1 | strptime("%Y-%m-%d") | mktime)
            -
            ($d2 | strptime("%Y-%m-%d") | mktime)
          ) / 86400
          | floor
          catch null;

        .goals | [.[] |
          ((.milestones // []) | length) as $total |
          ([(.milestones // [])[] | select(.completed == true)] | length) as $done |
          (if $total > 0 then (($done * 100) / $total | floor) else 0 end) as $pct |
          days_between(.deadline; $today) as $days_left |
          # Find next incomplete milestone (earliest due date first)
          ([(.milestones // [])[] | select(.completed != true)] | sort_by(.due) | first) as $next_ms |
          # Days until next milestone
          (if $next_ms then days_between($next_ms.due; $today) else null end) as $ms_days |
          {
            name: .name,
            deadline: .deadline,
            progress_pct: $pct,
            completed: $done,
            total: $total,
            days_left: $days_left,
            next_milestone: ($next_ms.title // null),
            next_milestone_due: ($next_ms.due // null),
            next_milestone_days: $ms_days,
            status: (
              if $days_left == null then "ON_TRACK"
              elif $days_left < 0 then "OVERDUE"
              elif $days_left == 0 then "DUE_TODAY"
              elif $days_left <= 3 then "URGENT"
              elif $days_left <= $look then "SOON"
              else "ON_TRACK" end
            ),
            ms_status: (
              if $next_ms == null then "COMPLETE"
              elif $ms_days == null then "MS_NO_DUE"
              elif $ms_days < 0 then "MS_OVERDUE"
              elif $ms_days == 0 then "MS_TODAY"
              elif $ms_days <= $look then "MS_SOON"
              else "MS_OK" end
            )
          }
        ]
      ' "$tmpfile"

  - id: report
    stdin: $evaluate-progress.stdout
    command: |
      tmpfile=$(mktemp /tmp/lb_eval_XXXXXX.json)
      trap 'rm -f "$tmpfile"' EXIT
      cat > "$tmpfile"
      jq -r '
        def goal_icon:
          if .status == "OVERDUE" then "🔴"
          elif .status == "DUE_TODAY" then "🟠"
          elif .status == "URGENT" then "🟡"
          elif .status == "SOON" then "🔵"
          else "🟢" end;

        def ms_icon:
          if .ms_status == "COMPLETE" then "🏆"
          elif .ms_status == "MS_NO_DUE" then "❓"
          elif .ms_status == "MS_OVERDUE" then "🔴"
          elif .ms_status == "MS_TODAY" then "🟠"
          elif .ms_status == "MS_SOON" then "🟡"
          else "✅" end;

        def progress_bar:
          ((.progress_pct / 10) | floor) as $filled |
          (10 - $filled) as $empty |
          ("█" * $filled) + ("░" * $empty) +
          " \(.progress_pct)% (\(.completed)/\(.total))";

        if length == 0 then
          "📚 No learning goals configured yet.\nCreate a goals JSON file and try again!"
        else
          "📚 Learning Goal Tracker\n" +
          "═══════════════════════════════════\n\n" +

          # Urgent section: includes goals with overdue milestones even if goal deadline is distant
          def needs_attention: .status == "OVERDUE" or .status == "DUE_TODAY" or .status == "URGENT" or .ms_status == "MS_OVERDUE";
          (if [.[] | select(needs_attention)] | length > 0 then
            "🚨 NEEDS ATTENTION\n" +
            "───────────────────\n" +
            ([.[] | select(needs_attention) |
              "\(goal_icon) \(.name)\n" +
              "   Progress: \(progress_bar)\n" +
              "   Deadline: \(.deadline) — " +
              (if .days_left == null then "unknown deadline"
               elif .days_left < 0 then "OVERDUE by \(- .days_left) days!"
               elif .days_left == 0 then "DUE TODAY!"
               else "due in \(.days_left) days" end) + "\n" +
              (if .next_milestone then
                "   \(ms_icon) Next milestone: \(.next_milestone)" +
                (if .next_milestone_due then " (due \(.next_milestone_due))" else "" end) + "\n"
               else "" end)
            ] | join("\n")) + "\n"
          else "" end) +

          # On track section: excludes goals already shown in needs-attention
          (if [.[] | select(needs_attention | not)] | length > 0 then
            "✅ ON TRACK\n" +
            "───────────\n" +
            ([.[] | select(needs_attention | not) |
              "\(goal_icon) \(.name)\n" +
              "   Progress: \(progress_bar)\n" +
              "   Deadline: \(.deadline) — " +
              (if .days_left == null then "unknown" else "\(.days_left) days left" end) + "\n" +
              (if .next_milestone then
                "   \(ms_icon) Next milestone: \(.next_milestone)" +
                (if .next_milestone_due then " (due \(.next_milestone_due))" else "" end) + "\n"
               else "" end)
            ] | join("\n")) + "\n"
          else "" end) +

          # Summary
          "───────────────────────────────────\n" +
          "📊 Summary: \(length) goal(s) | " +
          "\([.[] | select(.status == "OVERDUE")] | length) overdue | " +
          "\([.[] | select(.status == "DUE_TODAY" or .status == "URGENT")] | length) needs attention | " +
          "\([.[] | select(.status == "ON_TRACK" or .status == "SOON")] | length) on track\n" +
          "📈 Average progress: \(([.[] | .progress_pct] | add / length | floor))%"
        end
      ' "$tmpfile"

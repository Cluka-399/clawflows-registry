# Test Coverage Monitor

Track test coverage trends and alert on drops below threshold.

## What It Does

- Runs test coverage commands and tracks results over time
- Alerts when coverage drops below a configured threshold
- Helps maintain code quality standards

## Requirements

| Capability | Description |
|-----------|-------------|
| `exec` | Run test coverage commands |

## Configuration

Configure the test command and coverage threshold in the automation config.

## Schedule

Runs on weekdays during work hours.

## Installation

1. Add this automation to your ClawFlows registry
2. Configure your test coverage command
3. Set your minimum coverage threshold
4. Deploy and monitor

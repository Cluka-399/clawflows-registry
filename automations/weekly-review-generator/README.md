# Weekly Review Generator

Generate a weekly summary from completed tasks, commits, and calendar events.

## What It Does

- Collects completed tasks, git commits, and calendar events from the past week
- Generates a structured weekly review
- Highlights accomplishments and upcoming priorities

## Requirements

| Capability | Description |
|-----------|-------------|
| `calendar` | Access calendar events |
| `exec` | Run git commands for commit history |

## Schedule

Fridays at 5pm (`0 17 * * 5`).

## Installation

1. Add this automation to your ClawFlows registry
2. Connect calendar and git repos
3. Deploy

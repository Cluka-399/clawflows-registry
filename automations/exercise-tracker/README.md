# Exercise Tracker

Log workouts, track streaks, and celebrate fitness milestones.

## What It Does

1. **Logs workouts** - Record exercise type, duration, intensity
2. **Tracks streaks** - Counts consecutive active days
3. **Celebrates milestones** - Notifies you on streak achievements

## Requirements

| Capability | Example Skills |
|------------|----------------|
| `cron` | cron |

## Usage

Trigger manually to log a workout, or set up a daily check for streak tracking.

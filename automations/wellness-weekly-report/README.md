# Wellness Weekly Report

Weekly health summary covering sleep, exercise, mood, and habits.

## What It Does

- Aggregates wellness data from the past week
- Summarizes sleep patterns, exercise, mood trends, and habit streaks
- Provides actionable health insights

## Requirements

No external capabilities required — reads from local data files.

## Schedule

Sundays at 9am (`0 9 * * 0`).

## Installation

1. Add this automation to your ClawFlows registry
2. Configure your wellness tracking data source
3. Deploy

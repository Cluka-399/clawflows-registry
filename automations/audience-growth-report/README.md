# Audience Growth Report

Weekly report on audience growth and demographics across your social platforms.

## What It Does

1. **Fetches metrics** - Pulls audience data from configured social platforms
2. **Tracks growth** - Compares week-over-week changes
3. **Generates report** - Summary with key growth metrics and trends

## Requirements

| Capability | Example Skills |
|------------|----------------|
| `web-search` | social platform APIs |
| `chart-image` | trend visualization |

## Configuration

```yaml
config:
  platforms: ["twitter", "linkedin"]
```

## Schedule

Weekly on Monday at 9am: `0 9 * * 1`

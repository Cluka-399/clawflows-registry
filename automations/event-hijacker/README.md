# Event Hijacker

Never miss a relevant event. Auto-register, submit CFPs, and show up prepared.

## What It Does

- Monitors EventBrite, Luma, Meetup, CFP boards
- Auto-registers for virtual events
- Submits CFP proposals using your speaker kit
- Researches speakers and attendees
- Generates custom talking points
- Schedules follow-up messages

> "You show up to events prepared. The agent already knows who's attending, what they care about, and what to say."

## Requirements

- `web_search`, `browser`, `file_system` capabilities
- Speaker kit (bio, headshot, talk abstracts)

# A/B Test Automator

Continuous optimization without manual intervention.

## What It Does

- Monitors active A/B tests
- Calculates statistical significance
- Auto-promotes winners when confident
- Generates new variations to test
- Creates continuous optimization loop

> "Continuous optimization without manual intervention."

## Requirements

- `web_fetch`, `file_system`, `messaging` capabilities
- Landing page builder API access

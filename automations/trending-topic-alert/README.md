# Trending Topic Alert

Alert when topics in your niche start trending.

## What It Does

- Monitors web search trends for configured topics
- Detects when topics gain traction
- Sends alerts so you can engage early

## Requirements

| Capability | Description |
|-----------|-------------|
| `web-search` | Search for trending topics |

## Configuration

Set your niche topics and alert thresholds.

## Schedule

Checks every 4 hours for trending topics.

## Installation

1. Add this automation to your ClawFlows registry
2. Configure your topics of interest
3. Set notification preferences
4. Deploy

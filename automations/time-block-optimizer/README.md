# Time Block Optimizer

Analyze calendar patterns and suggest optimal meeting-free blocks for deep work.

## What It Does

- Analyzes your calendar for meeting patterns
- Identifies available focus time blocks
- Suggests optimal scheduling for deep work sessions

## Requirements

| Capability | Description |
|-----------|-------------|
| `calendar` | Access calendar events |

## Configuration

Set your preferred focus block duration and working hours.

## Schedule

Runs weekly on Monday mornings to plan the week.

## Installation

1. Add this automation to your ClawFlows registry
2. Connect your calendar
3. Configure preferred focus block duration
4. Deploy

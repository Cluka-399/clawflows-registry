# Hashtag Researcher

Research trending and relevant hashtags for your niche to boost social media reach.

## What It Does

1. **Searches for trending hashtags** in your specified niche
2. **Analyzes relevance** and popularity
3. **Returns curated hashtag sets** for your posts

## Requirements

| Capability | Example Skills |
|------------|----------------|
| `web-search` | web-search |

## Configuration

```yaml
config:
  niche: "tech"          # Your content niche
  platform: "instagram"  # Target platform
```

## Install

```bash
openclaw install hashtag-researcher
```

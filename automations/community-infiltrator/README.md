# Community Infiltrator

Be in the right place at the right time. Every time.

## What It Does

- Monitors Slack workspaces, Discord servers, Facebook groups
- Detects "What's the best tool for X?" questions
- Drafts helpful (not salesy) responses
- Alerts you with context + suggested reply

> "You show up as a helpful community member who happens to have built the exact thing they need."

## Requirements

- `browser`, `messaging` capabilities
- Community access (invites/memberships)

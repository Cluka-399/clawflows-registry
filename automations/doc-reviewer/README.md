# Doc Reviewer

Upload a PDF or text document and get a plain-English summary with flagged clauses and key takeaways.

## What It Does

1. **Parses document** — PDF, DOCX, or plain text
2. **Generates summary** in plain English
3. **Flags important clauses** — deadlines, penalties, obligations
4. **Extracts key takeaways** for quick decision-making

## Requirements

| Capability | Example Skills |
|------------|----------------|
| LLM | Built-in (no external skills needed) |

## Usage

```bash
lobster run doc-reviewer --args-json '{"file": "contract.pdf"}'
```

## Install

```bash
lobster install doc-reviewer
```
